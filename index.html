hw-03: Interactive Webpage, and Marks & Channels

Marks

  • Areas
    • This visualization contains areas since each state is represented with a 2D height and width.
    • This was a good choice because it is accurate, discernable, separable, and groups well. Specifically, the reader can easily understand that this is a political map (accurate). The levels of channels are easily discernable as the reader can differentiate between color and position. We can also see that channels exist integrally with each other and distant groups are easily inferable.
  • Points
    • This visualization contains points since Nebraska and Maine's electoral votes are depicted with 2D squares (area does not matter).
    • This was a good choice because it is easily discernable from the rest of the map and does not confuse the reader. The colors are consistent to allow for easy grouping.

Channels

  • Position
    • The amount of color on the slider communicates number of votes need to win.
    • This was not a geat choice because it does not allow for strong accuracy. Although reading the numbers, a reader can understand votes needed to win, the color of the slider tells the opposite story. The color indicates that the democratic party is winning.
  • Color
    • Blue and red are used to represent the democratic and republican parties, respectively. In addition, a combination of blue and red are used to indicate that Nebraska had a combination of blue and red voters.
    • This is a good choice the colors follow the existing colors of the two parties. Most readers would not have to read the title of the map to recognize that this refers to U.S. politics. It has a high degree of effectiveness and has visual-popout.

Colormaps(s)

  • Categorical
    • This is a categorical color map because it classifies color into the two political parties.
    • This is a good choice because it only uses colors previously associated with the political parties and does not confuse the reader. It saves space on the encoding as well because there is no need for a legend.

Marks

  • Points
    • This visualization contains points since each batted ball is represented by a 2D circle (area does not matter) on the heat map.
    • This is good mapping since plotting every single batted ball allows the user to get a better sense of how many balls were hit at the different launch angles and exit velocities; creating a easily readable scatter.
  • Lines
    • This visualization contains lines since it uses horizontal lines to categorize which launch angles represent which types of balls.
    • This is good mapping because it allows for easy grouping and is easily discernable.

Channels

  • Position
    • As a 2D plane, the the position of the marks communicates Launch angle (Vertical) and Exit velocity (Horizontal) on the heat map.
    • This is a good mapping because the spatial placement is effective in communicating the efficacy of each mark.
  • Color
    • The color represents the scoring value of a ball.
    • This is a good mapping because each color category is easily discernable from the other. Although nuances between the saturation of each hue is more difficult to see, the differences between "Below", "Avg." and "Above Avg." is distinguishable.

Colormaps(s)

  • Divergent
    • This is a divergent colormap as it uses two colors to indicate the extreme ranges (Below average LWTS vs. Above average LWTS) with a neutral color to represent the average LWTS.
    • This is a good choice because the reader understand the scale of LWTS that visualization is trying to communicate.

Marks

  • Areas
    • This visualization contains areas since each country is represented with a 2D height and width.
    • This was a good choice because it is accurate, discernable, separable, and groups well. Specifically, the reader can easily understand that this is a world map (accurate). The levels of channels are easily discernable as the reader can differentiate between color and position. We can also see that channels exist integrally.

Channels

  • Position
    • The spatial of the position of each country shows their relative distance from one another and communicates that this is a world map.
    • This is good choice as it is the most accurate way of visualization a world map.
  • Color
    • The color represents the average happiness score of a country.
    • This is a good mapping because the user can get all the information at once while also being able to differentiate between each country's happiness score. It is easy to understand the accuracy due to the changing saturation of the colors.

Colormaps(s)

  • Sequential
    • This is a sequential colormap as it uses saturation to indicate the increasing or decreasing amount of happiness from one country to the next.
    • This is a good choice because it categorizes the scale of happiness and allows the reader to discern whether a country's happiness score is low or high depending on saturation.
